# Интерпретатор лямбда-исчисления

Проект выполнен в рамках практики по курсу **"Математическая логика и теория алгоритмов"** (ФПМИ МФТИ, Петров Владислав Б05-515).

## Синтаксис модели
Вычислитель поддерживает следующий синтаксис:

* **Переменные:** строчные латинские буквы (`x`, `y`, `z`).
* **Абстракция:** `\var.body` (`\x.x`).
* **Аппликация:** `(M N)`.

## Как запустить
Для работы используется `uv`.

1. Установите зависимости: `uv sync`
2. Запустите интерпретатор: `uv run main.py <путь_к_файлу> --steps <число>`

### Параметры командной строки

* `file`: Путь к файлу с кодом для вычислителя. 

* `--steps`: Лимит шагов редукции (по умолчанию 1000). При превышении считается бесконечным циклом. 

* `-v, --verbose`: Включает явное логирование каждого шага редукции.
